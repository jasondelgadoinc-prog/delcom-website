- ✨ Modern, responsive design
- 🎨 Professional branding with DelCom logo
- 📱 Mobile-friendly layout
- 🚀 Fast-loading single-page site
- 💬 Contact form for lead generation
- 🎯 Service showcase
- 📊 About section

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## How to Use

1. Download or clone this repository
2. Open `index.html` in your browser
3. Customize with your own content as needed

## Deployment Options

### Option 1: Deploy to Netlify
1. Go to https://app.netlify.com
2. Click "Add new site" → "Deploy manually"
3. Drag the entire folder to Netlify
4. Link your custom domain (delcom.live)

### Option 2: Deploy to GitHub Pages
1. Push to GitHub
2. Go to Settings → Pages
3. Select main branch as source
4. Your site will be live at username.github.io

### Option 3: Use Squarespace
1. Go to https://www.squarespace.com
2. Point your delcom.live domain to your Squarespace site
3. Follow Squarespace's domain connection guide
